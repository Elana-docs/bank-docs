# Обновление Nova Bank

При наличии обновлений приложение Nova Bank отправит вам уведомление с предложением обновить версию.

1. Нажмите **Обновить**.  
   Запустится процесс загрузки новой версии, и после завершения загрузки появится уведомление.

2. Нажмите **Установить**.

Если вы хотите отложить обновление, нажмите **Не сейчас** в уведомлении с предложением обновить приложение.

:::info
Уведомление появится повторно через неделю, пока приложение не будет обновлено.
:::

Чтобы обновить отложенную версию Nova Bank:

1. Нажмите на иконку Nova Bank на домашнем экране вашего устройства.
2. Перейдите в раздел **Мой профиль** в нижнем меню.
3. Нажмите **Обновить**.

Если установленная на вашем устройстве версия Nova Bank больше не поддерживается, приложение обновится автоматически с помощью механизма принудительного обновления.
